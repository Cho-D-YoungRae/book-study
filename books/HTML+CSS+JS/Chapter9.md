# Chapter 9. 자바스크립트 기초 문법

## Note

### 새로운 변수 선언 키워드 `let`

- 변수를 선언하는 키워드 `var`, `let` -> `let` 이 새로 추가된 것. 이것을 사용하자.
- 변수명 중복 불가능
- 호이스팅 불가능
- 스코프 범위 다름

### 상수 변수 `const`

- 기본적인 특징은 `let`와 같으나 상수

### 템플린 문자열

- \` 로 감싸여져 있다.
  - \`템플릿 문자열\`
- 문자열 내에서 개행이 실제로도 적용
- ${} 를 통해서 변수를 대입할 수 있다.
- 기존 문자열의 단점을 모두 개선한 것이므로 주로 이용된다.
